# Sales Data Analysis Dashboard
# Project Overview
This project analyzes electronic sales data to identify revenue trends, customer purchasing behavior, and product performance. A Power BI dashboard was created to visualize key business insights.
# Objective
- Analyze monthly sales trends
- Identify top-performing products
- Determine peak sales hours
- Discover high-revenue cities
# Dataset
The dataset contains sales transaction data including:
- Order ID
- Product
- Quantity Ordered
- Price Each
- Order Date
- City
- Month
- Sales
- Hour
Data cleaning steps:
- Removed duplicates
- Handled missing values
- Standardized column names
- Converted data types
# Tools Used
- Python (Pandas)
- Power BI
- Excel
- GitHub
